Global Tsunami Model

Website

This is the future home of the GTM website.

The Global Tsunami Model Network aims to do something similar to the Global Earthquake Model and the Global Volcano Model, with a focus on Tsunami modeling and hazard assessment.

  • Anything pushed to the GitHub repository GlobalTsunamiModel/GlobalTsunamiModel.github.io will appear on these pages. This is currently the landing page for the domain globaltsunamimodel.org.

  • The planning repository has wiki pages and Issues that might be used for discussion of future plans.

  • For other repositories, see the GlobalTsunamiModel Organization on GitHub.

  • The GlobalTsunamiModel Google Group can also be used for discussion.

  • For more information, see these notes, slides, posters, etc.
